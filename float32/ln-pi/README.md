# FLOAT32_LN_PI

> Natural logarithm of the mathematical constant [π][pi] as a single-precision floating-point number.

<section class="usage">

## Usage

```javascript
var FLOAT32_LN_PI = require( '@stdlib/constants/float32/ln-pi' );
```

#### FLOAT32_LN_PI

Natural logarithm of the mathematical constant [π][pi] as a single-precision floating-point number.

```javascript
var bool = ( FLOAT32_LN_PI === 1.1447298526763916 );
// returns true
```
