# FLOAT32_LN2

> [Natural logarithm][@stdlib/math/base/special/lnf] of `2` as a single-precision floating-point number.

<section class="usage">

## Usage

```javascript
var FLOAT32_LN2 = require( '@stdlib/constants/float32/ln-two' );
```

#### FLOAT32_LN2

[Natural logarithm][@stdlib/math/base/special/lnf] of `2` as a single-precision floating-point number.

```javascript
var bool = ( FLOAT32_LN2 === 0.6931471824645996 );
// returns true
```
