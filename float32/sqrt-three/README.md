# FLOAT32_SQRT3

> [Square root][@stdlib/math/base/special/sqrtf] of `3` as a single-precision floating-point number.

<section class="usage">

## Usage

```javascript
var FLOAT32_SQRT3 = require( '@stdlib/constants/float32/sqrt-three' );
```

#### FLOAT32_SQRT3

[Square root][@stdlib/math/base/special/sqrtf] of `3` as a single-precision floating-point number.

```javascript
var bool = ( FLOAT32_SQRT3 === 1.7320507764816284 );
// returns true
```
